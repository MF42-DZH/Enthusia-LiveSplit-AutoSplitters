# Enthusia LiveSplit Auto Splitter Scripts

Currently, these scripts only work for the NTSC-U version of Enthusia: Professional Racing, via the PCSX2 emulator.

## Usage

1. Locate your LiveSplit install folder.
2. Put "emu-help-v3" in the "Components" subfolder of the LiveSplit install folder.
3. Open PCSX2.
4. Open the "Edit Layout" window, add a "Scriptable Auto Splitter" component.
5. Open "Layout Settings", go to "Scriptable Auto Splitter", and load in the auto splitter file of the category you are doing.
6. Ensure "Start" and "Split" are enabled on the "Scriptable Auto Splitter" settings.

## Notes

1. MAKE SURE PCSX2 IS OPEN BEFORE OPENING LIVESPLIT. IF YOU'VE OPENED LIVESPLIT BEFORE OPENING PCSX2, CLOSE LIVESPLIT AND REOPEN IT.
2. USE THE SPLIT FILES PROVIDED HERE OR IN SRDC, DO NOT ADD OR REMOVE SPLITS, BUT YOU CAN RENAME THEM AND CHANGE ICONS. (Note that these split files were originally designed for the "Subsplits" layout component.)
